The Clean Architecture Solution Template gives you a production-ready starting point for building enterprise .NET applications. It wires together Clean Architecture's layered structure — Domain, Application, Infrastructure, and Web — with CQRS via MediatR, FluentValidation, Entity Framework Core, ASP.NET Core Identity, and .NET Aspire for local orchestration and testing.

Generate a new solution in one command, choosing your client framework (Angular, React, or API-only) and database provider (SQLite, PostgreSQL, or SQL Server). Everything is pre-wired and ready to run.

## What's included

The template generates a complete solution with the following structure:

| Project | Purpose |
|---|---|
| `Domain` | Entities, value objects, domain events, enums |
| `Application` | CQRS handlers, validators, pipeline behaviours |
| `Infrastructure` | EF Core DbContext, Identity, database migrations |
| `Web` | ASP.NET Core minimal API endpoints, SPA host |
| `AppHost` | .NET Aspire orchestration for local development |
| `ServiceDefaults` | OpenTelemetry, health checks, service discovery |

## Quick install

```bash
dotnet new install Clean.Architecture.Solution.Template
dotnet new ca-sln --client-framework angular --database sqlite --output MyApp
dotnet run --project MyApp/src/AppHost
```

<Tip>
  Run `dotnet new ca-sln --help` to see all available template options including client framework and database provider choices.
</Tip>
